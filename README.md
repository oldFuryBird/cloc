# cloc
## 代码统计器 0.1版本
### 已经实现的功能
   1. 统计代码文件行数
   2. 输出结果

### 即将实现的功能
   1. 过滤文件夹,文件     --exclude
   2. 指定文件后缀类型    --include-ext
   3. 注释不纳入统计      --no-comment
   4. 更多输出方式        --output  
   5. ...
